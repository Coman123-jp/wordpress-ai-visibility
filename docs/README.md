## Quick Reference

### Plugin settings location
WordPress admin → **AI Visibility** (sidebar, below Appearance)

### Option key prefix
All settings are stored in `wp_options` with the prefix `aiv_`.

### llms.txt URL
`https://yourdomain.com/llms.txt`

### REST API base
`/wp-json/ai-visibility/v1/`

### PHP namespace
`AIVisibility\`
